# Toy Robot Simulator

* Ruby and Rails version
    * Ruby version: 2.2.1
    * Rails version: 4.2.7

* Setting up development
    * Clone the project from github
        ````
        git clone https://github.com/sumitdey035/robot.git
        ````
    * Go to the project folder
        ````
        cd robot
        ````
    * Run bundler
        ````
        bundle install
        ````
    * Setup database
        ````
        rake db:setup
        ````
        
* Running development env:
    * Start rails server
        ````
        rails s
        ````
* Run specs
    ````
    bin/rspec spec
    ````
